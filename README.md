# Affiliate Blog — Spok HQ

Auto-generated SEO content via Ollama llama3.2:3b.
Deployed to GitHub Pages via Hugo static site.

## Stack
- Hugo (static site generator)
- GitHub Pages (free hosting)
- Ollama (local content generation, runs overnight)
- Amazon Associates + Google AdSense (monetization)

## Setup (after Hugo install)
1. `hugo new site .`
2. Pick a niche, set in config.toml
3. Run `./generate.sh` to generate articles
4. Push to GitHub → auto-deploys
